WWAN Settings

AUTOCONNECT:

Autoconnect feature sets the Auto Connect flag, the same is updated in the configuration file by Connection Manager.

ROAMING:

Roaming feature will enable MobileAP to connect to the network in roaming mode.

To save and cancel changes

Click UPDATE to save the new setting. Click REFRESH to discard any unsaved changes.
